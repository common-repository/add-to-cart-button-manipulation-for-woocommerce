WooCommerce extension that allows you to easily control "Add to Cart" button on single product page. 

== Description ==
WooCommerce extension that allows you to easily control "Add to Cart" button on single product page. You can allow or disallow "Add to Cart" button with just one click or specify
time frame inside which is product purchasable (with nice countdown). That way you can have some special offers or limited time offers for your clients and shoppers.
